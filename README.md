# Hacker News Clone

The project was created while following Scrimba's Javascript Deep Dive course. 
It replicates key features of the original site, such as displaying top stories, comments, and navigation (using Navigo). 

The course covered:
  Types & Conditionals
  Functions
  Objects & Maps
  Arrays & Sets
  Classes
  DOM
  Async Javascript
